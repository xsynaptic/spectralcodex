# Development Commands

## Core Commands

- `pnpm dev` - Start development server
- `pnpm build` - Build for production (includes type checking); may take 10+ minutes
- `pnpm preview` - Preview production build
- `pnpm check-types` - Run TypeScript type checking
- `pnpm lint` - Run ESLint
- `pnpm test` - Run tests

## Content Commands

- `pnpm content-related` - Generate related content links
- `pnpm content-schemas` - Generate content schemas
- `pnpm content-validate` - Validate content
- `pnpm lint-content` - Lint content package
- `pnpm lint-content-demo` - Lint content-demo package

## Utility Commands

- `pnpm map-divisions` - Extract administrative boundaries from Overture Maps
- `pnpm map-spritesheet` - Generate map icon spritesheet
- `pnpm opengraph-image` - Generate OpenGraph images
- `pnpm deploy-static` - Deploy static build; may take 10+ minutes
